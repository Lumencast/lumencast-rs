# Code of Conduct

This project adopts the [Contributor Covenant v2.1][cc].

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, assume good faith, focus on the work. Harassment,
insults, or personal attacks are not tolerated.

Report concerns to the maintainers at the address listed in
[SECURITY.md](SECURITY.md).
